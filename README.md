# IELTS Claude Code Skills V2.0

> 用 AI 代替雅思培训班。6 个 Claude Code Skill 组成一个完整的备考教练团队——诊断定位、写作批改、阅读精读、口语素材、词汇训练、进度追踪，全部跨会话记忆。

### 这套系统做了什么

| | 传统方式 | 这套系统 |
|--|---------|---------|
| 写作批改 | 找老师 ¥100-200/篇，等1-3天 | `/ielts-writing` 5分钟出四维评分+改写对比 |
| 备考规划 | 报班 ¥5000-20000 | `/ielts-diagnose` 根据成绩数据算出最高效提分路径 |
| 口语陪练 | 外教 ¥200-500/小时 | `/ielts-speaking` 生成素材 + Gemini Live 24h模拟考官 |
| 词汇训练 | 自己背单词书 | `/ielts-vocab` 间隔重复+同义替换专项训练 |
| 进度追踪 | 靠自觉 | `/ielts` 读取全部数据自动生成进度报告 |

### 我的成绩

```
听力 8.0 ｜阅读 8.0 ｜写作 7.0 ｜口语 7.0 ｜总分 7.5
备考 2 个月 ｜每天 2-3 小时 ｜
```

### 核心设计

- **6 个 Skill** — 教练/诊断/写作/阅读/口语/词汇，各司其职
- **持久化数据** — `~/.ielts/` 目录跨会话保留所有学习数据（批改记录、错题追踪、同义替换库、词汇进度）
- **SOUL 人格** — 每个 Skill 有独立人格设定（写作=考官级严格、词汇=Duolingo风格、教练=数字驱动）
- **数据驱动** — 教练读取全部 Skill 数据，用数字而不是感觉给建议

### 谁适合用

- 自学备考雅思，不想报班
- 会用 Claude Code（或愿意花10分钟装好）
- 目标 6.5-8.0，有2-3个月备考时间

## V2.0 重大更新

> 2026-04-08 发布。从「5个独立 Skill」升级为「6个 Skill + 共享数据层」。

### 新增

- **`/ielts-vocab`** — 全新词汇训练 Skill
  - 间隔重复：Day1 → Day2 → Day4 → Day7 → Day14
  - 同义替换专项训练（阅读核心能力）
  - 搭配练习（make/do/take/have）
  - 难词池自动管理（错1次进池，连对3次出池）
- **`~/.ielts/` 持久化数据系统** — 首次使用任意 Skill 自动创建文件夹结构，所有学习数据跨会话保留
- **进度报告模式** — 对 `/ielts` 说"看看进度"，教练读取全部数据生成报告+建议

### 升级

| 改动 | 说明 |
|------|------|
| SOUL 人格系统 | 每个 Skill 有独立人格：写作=考官级严格，词汇=Duolingo风格，教练=数字驱动 |
| 写作错误追踪 | `errors.md` 跨会话记录高频错误，连续3篇不犯标记为已解决 |
| 阅读同义替换库 | `synonyms.md` 每次分析追加新替换对，越用越强 |
| 口语素材持久化 | 话题分组+万能故事保存到文件，不用每次重新生成 |
| 诊断历史对比 | `scores.md` 记录分数变化，复诊自动对比上次 |
| 教练升级 | 从简单路由器升级为总教练，读取所有数据给数据驱动建议 |

### V1 vs V2

```
V1: 5个独立 Skill → 每次对话从零开始 → 无进度追踪
V2: 6个 Skill + ~/.ielts/ 数据层 → 跨会话记忆 → 数据驱动备考
```

## 各 Skill 详细使用

### `/ielts` — 教练

> 总指挥。路由 + 进度追踪 + 建议。

**触发词：** `/ielts`、「我要备考雅思」、「看看进度」、「我该练什么」

**首次使用：**
```
你：我要备考雅思
教练：你的目标分数是多少？考试什么时候？
你：目标7.5，6月考
教练：做过模考吗？四科分别多少？
你：L7 R6.5 W5.5 S6
教练：→ 路由到 /ielts-diagnose
```

**有数据后：**
```
你：看看进度
教练：
  写作：最近3篇平均6.0，TR持续偏低
  阅读：正确率从65%涨到78%，T/F/NG仍是主要错题
  词汇：Day 12，已掌握87词，难词池23词
  → 今天建议重点练写作，专攻论点展开
```

---

### `/ielts-diagnose` — 诊断

> 算账，不鼓励。找到最高效的提分路径。

**触发词：** `/ielts-diagnose`、「分析成绩」、「帮我做个计划」

```
你：L7 R6.5 W5.5 S6，目标7.5，两个月
诊断：
  推荐目标组合：L8 R8 W6.5 S6.5 = 7.25 → 7.5
  各科差距：听力+1 阅读+1.5 写作+1 口语+0.5
  → 80%时间给听力阅读，20%给写作口语
  → 输出8周计划 + 每日时间分配
```

**自动写入：** `profile.md` + `scores.md`。复诊自动对比历史。

---

### `/ielts-writing` — 写作批改（价值最大）

> 考官级四维批改 + 改写对比。

**触发词：** `/ielts-writing`、「批改作文」、「审题」、「写作练习」

| 模式 | 你给什么 | 它做什么 |
|------|---------|---------|
| 审题 | 题目 | 题型分类 + 关键词标注 + PEEL提纲 + 审题陷阱 |
| 批改 | 题目 + 作文 | 四维评分 + 逐句标注 + 改写对比 + 提分优先级 |
| 练习 | "给我一道题" | 出题 → 你写 → 批改 |

**批改示例：**
```
你：[贴题目 + 作文]
写作：
  TR 5.5 | CC 6.0 | LR 5.5 | GRA 6.0 | 总分 6.0

  逐句标注：
  > "Many people think technology has a bad effect..."
  > TR: 直接抄了题目。改为：Technology's influence on modern society...
  > LR: "bad effect" 太基础。改为：detrimental impact

  改写对比（6分 → 7分）：
  [你的版本 vs 改写版本，标注每处修改原因]

  提分优先级：1. TR论点展开 2. LR同义替换 3. GRA句型多样化
```

**跨会话追踪：** 下次批改时：「你最近3篇平均5.5，TR一直卡在5，论点展开是主要问题。这次重点看这个。」

---

### `/ielts-reading` — 阅读精读

> 教你怎么找到答案，不是告诉你答案。

**触发词：** `/ielts-reading`、「分析阅读」、「这道为什么错」、「同义替换」

| 模式 | 你给什么 | 它做什么 |
|------|---------|---------|
| 错题分析 | 文章 + 题目 + 你的答案 | 逐题拆解 + 同义替换词表 |
| 精读训练 | 文章 + 题目（没做过） | 引导做题，不给答案 |
| 专项训练 | "练T/F/NG" | 针对特定题型强化 |

**T/F/NG 三步框架：**
1. 原文有没有提到这个话题？→ 没提到 = NOT GIVEN
2. 提到了 → 意思一致 = TRUE / 意思矛盾 = FALSE
3. 提到了但信息不够判断 = NOT GIVEN

**同义替换库：** 每次分析追加到 `synonyms.md`，跨会话累计。越做越大。

---

### `/ielts-speaking` — 口语素材

> 5个万能故事覆盖80%话题。

**触发词：** `/ielts-speaking`、「口语素材」、「话题分组」、「万能故事」

| 模式 | 你给什么 | 它做什么 |
|------|---------|---------|
| 话题分组 | 题库或"帮我分组" | 50话题分5组 + 覆盖映射表 |
| 故事生成 | "准备旅行话题" | Part 2回答 + Part 3追问预测 |
| 表达升级 | 你的回答 | 词汇升级，保持口语感 |

**5组万能故事：** 旅行/地点、人物、物品/技能、经历/事件、媒体/学习

**重要：** 这个 Skill 只生成素材。练口语去 Gemini Live 或 ChatGPT Voice。

---

### `/ielts-vocab` — 词汇训练（V2 新增）

> Duolingo 猫头鹰风格。短、快、催你。

**触发词：** `/ielts-vocab`、「背单词」、「考我」、「练同义替换」、「练搭配」

| 模式 | 触发 | 做什么 |
|------|------|--------|
| 今日词汇 | 「背单词」 | 推15词（AWL 5 + 听力5 + 同义替换5） |
| 复习测试 | 「考我」 | 最近3天词汇抽10题 |
| 同义替换 | 「练同义替换」 | 给题目用词，你说替换词 |
| 搭配练习 | 「练搭配」 | make/do/take/have 选择题 |
| 难词复习 | 「复习难词」 | 从难词池抽题 |

**间隔重复：** Day1→Day2→Day4→Day7→Day14。答错进难词池，连对3次出池。

**测试示例：**
```
你：考我
词汇：significant 的中文意思？
你：重要的
词汇：对。accommodation 怎么拼？
你：accomodation
词汇：不是。accommodation，双c双m。进难词池，后天再考。
```

## 完整备考流程

```
第1天
└── 做一套剑桥真题（推荐C14-C19）
└── /ielts-diagnose → 诊断报告 + 8周计划
└── 自动创建 ~/.ielts/ 数据目录

每天（2小时）
├── 听力 60min — 精听 + 王陆语料库
├── 阅读 40min — 做题 + /ielts-reading 错题分析
├── /ielts-vocab 背词 15min
└── 口语 5min — 复习万能故事

每3天
└── 写一篇 Task 2 → /ielts-writing 四维批改

每周日
└── /ielts 「看看进度」→ 进度报告 + 下周建议

考前2周
├── /ielts-speaking 生成5个万能故事
└── Gemini Live 模拟口语考试

8周后 → 上考场
```
